# SpeedrunTools ビルド手順書
## プログラム知識ゼロでも大丈夫！

---

## 必要なもの（無料）

| ソフト | 用途 | ダウンロード先 |
|--------|------|----------------|
| Java 8 (JDK) | ビルドに必要 | https://adoptium.net/ → Java 8 (LTS) を選んでインストール |
| IntelliJ IDEA Community | ビルドボタンを押すだけ | https://www.jetbrains.com/idea/download/ → Community版（無料）|

---

## Forge版のビルド手順

### 1. フォルダを開く
1. `speedruntools` フォルダを好きな場所に置く
2. IntelliJ IDEA を起動
3. **「Open」** をクリック → `speedruntools` フォルダを選択 → **「OK」**

### 2. Forgeのセットアップ（初回のみ）
1. IntelliJ の下部に **「Terminal」** タブがある（なければ View → Tool Windows → Terminal）
2. 以下をコピーして貼り付けて Enter：
   ```
   gradlew genIntellijRuns
   ```
   ※ダウンロードが始まります。5〜10分待つ

### 3. ビルドする
1. 右側の **「Gradle」** パネルをクリック
2. `speedruntools` → `Tasks` → `build` → **`build`** をダブルクリック
3. 下部に **「BUILD SUCCESSFUL」** と出たら完成！

### 4. .jarファイルの場所
```
speedruntools/build/libs/speedruntools-1.0.0.jar
```
このファイルを Minecraft の **mods フォルダ** に入れる。

## Fabric版のビルド手順

### 1. フォルダを開く
1. `speedruntools-fabric` フォルダを好きな場所に置く
2. IntelliJ IDEA を起動
3. **「Open」** をクリック → `speedruntools-fabric` フォルダを選択 → **「OK」**

### 2. ビルドする
1. 右側の **「Gradle」** パネルをクリック
2. `speedruntools-fabric` → `Tasks` → `build` → **`build`** をダブルクリック
3. **「BUILD SUCCESSFUL」** と出たら完成！

### 3. .jarファイルの場所
```
speedruntools-fabric/build/libs/speedruntools-fabric-1.0.0.jar
```

---

## Minecraftへの入れ方

### Forge版
1. Minecraft Launcher で **Forge 1.16.1** プロファイルを選択
2. `%appdata%\.minecraft\mods` フォルダを開く
   （Windowsキー + R → `%appdata%\.minecraft\mods` と入力）
3. `speedruntools-1.0.0.jar` をそのフォルダに入れる
4. Minecraft を起動

### Fabric版
1. Fabric Loader 1.16.1 をインストール（https://fabricmc.net/use/）
2. Fabric API も mods フォルダに入れる（必須）
   ダウンロード: https://www.curseforge.com/minecraft/mc-mods/fabric-api
3. `speedruntools-fabric-1.0.0.jar` を mods フォルダに入れる
4. Minecraft を起動

## Mod の使い方

| キー | 機能 |
|------|------|
| **O キー** | SpeedrunTools 設定画面を開く |
| **C キー（長押し）** | ズーム。ホイールで拡大率調整 |

### 設定画面の項目

#### 暗視
- オン/オフ で切り替え
- オンにすると常時明るい状態（Night Visionポーション不要）

#### タイマー
1. 「タイマー」を押す
2. 止める条件を選ぶ（複数選択可）
   - ✓ エンドラが死んだ瞬間に止まる
   - ✓ エンドポータルに入った瞬間に止まる
   - ✓ どちらも選択しない（手動）
3. 「決定」を押す
4. 「スタート」を押してゲームに戻る
5. 最初に動く・ブロックを壊す などのアクションでタイマー自動スタート
6. 右上に黄色でタイマーが表示される

#### 現世とネザーの距離計測
- ＋／－ボタンで座標を入力
- 方向を「現世→ネザー（÷8）」「ネザー→現世（×8）」で切り替え
- 結果がリアルタイムで表示

#### エンド要塞特定（NinjabrainBot方式）
1. Minecraft の **設定 → FOVを30** にする
2. エンダーアイを投げる
3. 飛んでいくアイに視点をぴったり合わせる
4. **F3 + C** を押す（自動でデータ取得）
5. 100ブロック以上離れた場所でもう一度繰り返す
6. 設定画面 → 「エンド要塞特定」でX・Z座標と誤差が表示される

## エラーが出たとき

| エラー | 解決策 |
|--------|--------|
| `JAVA_HOME is not set` | Java 8 JDK を再インストールしてPCを再起動 |
| `BUILD FAILED` | IntelliJ の Terminal で `gradlew clean` を実行してからもう一度 build |
| ゲーム起動時にクラッシュ | Forge/Fabricのバージョンが1.16.1か確認 |
